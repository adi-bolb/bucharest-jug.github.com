Bucharest JUG 3rd edition

You are welcome to participate at the 3rd edition of Bucharest Java User Group.

The presentations for this editions are as follows:

  • Fast guide to JAAS by Florin Buzatu
  • Trove - memory efficient data structures by Corneliu Creanga
  • Code Review for Secure Web Application by Silvia Dumitrescu

The 3rd meeting will be held at Adobe Romania headquarters starting from 19:00 on July 26'th. The adress is: Anchor Plaza, 9th floor, Bd. Timisoara, Nr. 26 Z, Bucharest, District 6, Zip code 061331, near Mall Plaza Romania
