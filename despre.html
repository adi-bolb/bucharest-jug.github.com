Istoric

Grupul utilizatorilor de Java din București (JUG București) a apărut pentru a crea o comunitate a dezvoltatorilor ce folosesc platforma Java.

Grupul a luat ființă la inițiativa lui Andrei Savu. Membrii "fondatori" sunt Andrei Savu, Ioan Eugen Stan și Mihai Soloi. Prima ediție a avut loc la sediul Ceata și au participat ~ 25 persoane. De acolo evenimentul a luat, încet, încet amploare.

Scop

JUG București își propune să creze o comunitate puternică a dezvoltatorilor de Java, Groovy, Clojure și alte limbaje ce folosesc ca suport mașina virtuală Java (JVM).

Dorim să realizăm scopul prin organizarea unor întâlniri periodice cu prezentări despre tehnologii din universul Java și ieșire la bere pentru socializare.
