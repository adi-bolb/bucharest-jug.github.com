Bucharest JUG 1'st edition

You are welcome to participate at the first edition of Bucharest Java User Group.

The presentations for this first editions are as follows:

  • Google Guava Overview by Andrei Savu (level: intermediate Java users)
  • Apache Maven by Mihai Soloi(level: beginer/intermediate)
  • Apache James: more than emails in the cloud by Ioan Eugen Stan (level:intermediate - prelude for Berlin Buzzwords)

The first meeting will be held at Ceata's headquarters (Vatra) starting from 19:00 on May 24'th. The adress is: 39A Corneliu Coposu, ap. 3. Close to Unirii Square.

Prima ediție a JUG București

Vă invităm să participați la prima întâlnire a Grupului Utilizatorilor de Java din București.

Prezentările pentru prima ediție sunt după cum urmează:

  • Prezentare generală Google Guava ținută de Andrei Savu (nivel: utilizatori intermediari)
  • Apache Maven ținută de Mihai Soloi(nivel: începtăori/intermediari)
  • Apache James: more than emails in the cloud ținută de Ioan Eugen Stan (nivel: intermediarî - exercițiu înainte de Berlin Buzzwords)

Prima întâlnire va avea loc la Vatră, sediul Ceata, începând cu ora 19:00 pe data de 24 mai. Adresa este: 39A Corneliu Coposu, ap. 3. Aproape de Piața Unirii (5 minute de mers).
